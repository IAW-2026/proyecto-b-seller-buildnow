# Build Now - Seller App

Aplicación **Seller** del [Proyecto IAW 2026](https://iaw-2026.github.io/proyecto/). 

Esta plataforma permite a los vendedores gestionar su tienda, catálogo de productos y órdenes, y a los administradores moderar la plataforma en general (tiendas, categorías, órdenes globales).

Para probar las funcionalidades de la aplicación, puedes acceder con los siguientes usuarios de prueba:

### 1. Administrador (Admin)
Gestiona tiendas, categorías y órdenes a nivel global (solo puede verlas).

- **Contraseña:** `plan3000plan3000`

### 2. Vendedor (Seller)
Gestiona su propia tienda, inventario y ventas.
- **Email:** `fede@gmail.com`
- **Contraseña:** `plan3000plan3000`
*(Nota: Si no hay credenciales de prueba, puedes registrarte como nuevo vendedor directamente en la aplicación).*
